### Examples

**Note** `#` Means Having Same Values

- NAND -> #NAND => AND

- ( #NAND - #NAND ) -> NAND => OR

- #NAND => NOT

- ( #NAND - #NAND ) -> NAND -> #NAND => NOR

- NOR -> #NOR => OR

- ( #NOR - #NOR ) -> NOR => AND

- #NOR => NOT

- ( #NOR - #NOR ) -> NOR -> #NOR => NAND
